---
description: Processo completo para desenvolver e entregar novas funcionalidades
---

1. [Especificação]
   - Definir o escopo da feature.
   - Listar os requisitos funcionais e não funcionais.
   - (Opcional) Criar mockups ou diagramas.

2. [Planejamento Técnico]
   - Usar o comando `/plan`.
   - Identificar impacto na arquitetura existente.

3. [Implementação]
   - Codificar a funcionalidade seguindo os padrões do projeto.
   - Criar novos arquivos ou módulos se necessário.

4. [Testes]
   - Criar testes unitários ou de integração.
   - Realizar validação funcional.

5. [Documentação]
   - Atualizar README.md ou criar documentação específica se a feature for complexa.
