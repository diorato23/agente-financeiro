description: Planejamento e execução de testes sistemáticos
---

1. [Seleção]
   - Quais componentes foram alterados?
   - Quais tipos de teste são necessários (Unitário, Integração, E2E)?

2. [Preparação]
   - Configurar massa de dados de teste.
   - Garantir ambiente de teste isolado.

3. [Execução]
   - Rodar testes automatizados (pytest, jest, etc).
   - Executar roteiros de teste manual.

4. [Reporte]
   - Registrar falhas e sucessos.
   - Documentar bugs encontrados para correção posterior.
