---
description: Design de interface e experiência de usuário de alta fidelidade
---

1. [Inspiração e Conceito]
   - Buscar referências visuais de alto nível (Dribbble, Awwwards).
   - Definir paleta de cores, tipografia e "mood".

2. [Wireframing (Baixa Fidelidade)]
   - Esboçar a estrutura da página.
   - Definir hierarquia de informações.

3. [Prototipagem (Alta Fidelidade)]
   - Criar o design visual detalhado.
   - Aplicar efeitos modernos (Glassmorphism, sombras suaves, gradientes).
   - Definir micro-interações e animações.

4. [Implementação CSS/Frontend]
   - Traduzir o design para código (CSS3, Tailwind, etc).
   - Garantir responsividade (Mobile First).

5. [Polimento]
   - Ajustar espaçamentos (padding/margin) com precisão de pixel.
   - Testar acessibilidade e contraste.
